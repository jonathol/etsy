# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Category

- `GET /api/category`
- `GET /api/category/:categoryId`

### Type

- `GET /api/categoy/:categoryId/:typeId`

### Specific

- `GET /api/categoy/:categoryId/:typeId/:specificId`

### Cart

- `GET /api/cart`
- `POST /api/cart/new`
- `DELETE /api/cart/:id/`

### Listing

- `GET /api/listing/:id`
