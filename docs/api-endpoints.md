# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Cart

- `GET /api/cart`
- `GET /api/cart/:id`
- `POST /api/cart/new`
- `DELETE /api/cart/:id`

### Listing

- `GET /api/listing`
- `GET /api/listing/:id`
